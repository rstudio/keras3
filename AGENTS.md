# AGENTS

Short notes for working in this repo:

- Use `Rscript tools/knit-vignettes.R` and `Rscript tools/knit-examples.R` to
  render guides/examples; `tools/knit.R` handles most setup.
- `tools/make-website.R` builds the pkgdown site into `docs/dev`.
- NEWS entries are expected for user-facing changes in `NEWS.md`.
