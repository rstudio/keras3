# Publicly accessible method for determining the current backend.

Publicly accessible method for determining the current backend.

## Usage

``` r
config_backend()
```

## Value

String, the name of the backend Keras is currently using. One of
`"tensorflow"`, `"torch"`, or `"jax"`.

## Examples

    config_backend()

    ## [1] "tensorflow"
