# Relu6 activation function.

It's the ReLU function, but truncated to a maximum value of 6.

## Usage

``` r
activation_relu6(x)
```

## Arguments

- x:

  Input tensor.

## Value

A tensor, the result from applying the activation to the input tensor
`x`.
