# Check whether NNX-specific features are enabled on the JAX backend.

Check whether NNX-specific features are enabled on the JAX backend.

## Usage

``` r
config_is_nnx_enabled()
```

## Value

Logical flag; `TRUE` if NNX backend features are enabled, `FALSE`
otherwise.
