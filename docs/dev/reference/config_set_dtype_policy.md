# Sets the default dtype policy globally.

Sets the default dtype policy globally.

## Usage

``` r
config_set_dtype_policy(policy)
```

## Arguments

- policy:

  A string or `DTypePolicy` object.

## Value

No return value, called for side effects.

## Examples

    config_set_dtype_policy("mixed_float16")
