# Returns predictions for a single batch of samples.

Returns predictions for a single batch of samples.

## Usage

``` r
predict_on_batch(object, x)
```

## Arguments

- object:

  Keras model object

- x:

  Input data. It must be array-like.

## Value

Array(s) of predictions.
