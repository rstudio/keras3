# Returns the indices of the bins to which each value in `x` belongs.

Returns the indices of the bins to which each value in `x` belongs.

## Usage

``` r
op_digitize(x, bins, zero_indexed = FALSE)
```

## Arguments

- x:

  Input array to be binned.

- bins:

  Array of bins. It has to be one-dimensional and monotonically
  increasing.

- zero_indexed:

  If `TRUE`, the returned indices are zero-based (`0` encodes to first
  bin); if `FALSE` (default), the returned indices are one-based (`1`
  encodes the first bin).

## Value

Output array of indices, of same shape as `x`.

## Examples

    x <- op_array(c(-1,    0, 1,    3, 3.5,      4.5,  1.6,    8))
    bins <- array(c(    0,       3,         4.5,            7))
    op_digitize(x, bins)

    ## tf.Tensor([1 2 2 3 3 4 2 5], shape=(8), dtype=int32)
