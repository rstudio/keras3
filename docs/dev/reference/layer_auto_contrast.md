# Performs the auto-contrast operation on an image.

Auto contrast stretches the values of an image across the entire
available `value_range`. This makes differences between pixels more
obvious. An example of this is if an image only has values `[0, 1]` out
of the range `[0, 255]`, auto contrast will change the `1` values to be
`255`.

This layer is active at both training and inference time.

## Usage

``` r
layer_auto_contrast(object, value_range = tuple(0L, 255L), ...)
```

## Arguments

- object:

  Object to compose the layer with. A tensor, array, or sequential
  model.

- value_range:

  Range of values the incoming images will have. Represented as a two
  number tuple written `(low, high)`. This is typically either `(0, 1)`
  or `(0, 255)` depending on how your preprocessing pipeline is set up.
  Defaults to `(0, 255)`.

- ...:

  For forward/backward compatability.
